NexTrade
A full-stack cryptocurrency trading platform built with Next.js, featuring database integration via Sequelize, user authentication, wallet management, and crypto utilities.
Features

Modern Next.js app (App Router) with Tailwind CSS/PostCSS for styling.
Backend database models and migrations (Sequelize) for users, trades, balances, etc.
Crypto tools: Key generation (generateKeys.mjs), address matching (addressmatch.js).
Static assets including Bitcoin imagery.
ESLint and VS Code configurations for development.
Environment and config management.

Tech Stack

Frontend/Backend: Next.js
Database: Sequelize (with migrations and models)
Styling: Tailwind CSS, PostCSS
Linting: ESLint
Other: Node.js scripts for crypto operations
